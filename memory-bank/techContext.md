# Technical Context: Closspad

## Technology Stack
- Language: ClojureScript
- UI: Replicant (React-like)
- Routing: Reitit
- Styling: Tailwind CSS and Daisy UI
- Async: core.async
- Backend: Supabase REST API

## Development Setup
- Build tool: Shadow CLJS
- Package manager: npm
- Configuration files:
  * shadow-cljs.edn
  * package.json
  * deps.edn

## Key Dependencies
- Replicant
- Reitit
- core.async
- Supabase client
- Tailwind CSS
